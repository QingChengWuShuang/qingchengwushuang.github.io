浅谈快速幂

Description

求 $ a^b $ mod p $( 0<a,b,p{\leq}10^9 )$



Solution

b可以表示为二进制数,所以 $ b=c_{k}2^{k}+c_{k-1}2^{k-1}+…+c_{0}2^{0} $,其中,$c_k$是b的二进制第k位的数字。

所以, $ a^b $ 可以表示为 $ a^{b}=a^{c_{k}2^{k}} {\times} a^{c_{k-1}2^{k-1}} {\times}…{\times}a^{c_{0}2^{0}} $
